# PersonalFinance-UWP

### Requirements:
- Windows 10 UWP SDK
- Visual Studio 2015 community or higher
- SQLite-net-pcl Nuget package
- Template 10

### Features
1. Multiple Accounts
2. Credit Ratio
3. Transactions
4. Debts
5. Assets
